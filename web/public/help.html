KAMADHENU WEB HELP
===================

$> kamadhenu
  Pull a fresh random fortune + random cow.

$> kamadhenu alias
  Put your own words in Kamadhenu's mouth.

$> kama help
  You're here.

$> kama source
  Open GitHub source.
